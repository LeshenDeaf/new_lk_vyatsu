## Начинаем

Чтобы запустить dev сервер на локалке нужно выполнить команду:

```bash
npm run dev
```

Нужно открыть [http://localhost:3000](http://localhost:3000) *НЕ 127.0.0.1:3000!* в браузере, чтобы увидеть результат.

[API пути](https://nextjs.org/docs/api-routes/introduction) располагаются по url [http://localhost:3000/api](http://localhost:3000/api). This endpoint can be edited in `pages/api/me.ts`.

Папка `pages/api` относится к `/api/*`. Файлы в этой папке выполняются как [API пути](https://nextjs.org/docs/api-routes/introduction), а не React страницы.
